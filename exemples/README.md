# Exemples de Polymers

## installation

Les répertoires node_modules et components n'ont pas été inclus pour alléger le clone.

lancer
```npm install```

pour lancer les exemples, il faut un serveur web. Les polymers sont importés en ajax, donc par soucis de sécurité, les browsers n'acceptent pas les cross-origin.
**polyserve** est un outil qui permet de lancer un mini serveur pour lancer les exemples.

```npm install -g polyserve```
pour l'installer globalement.
